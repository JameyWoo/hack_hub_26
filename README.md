# 体感鼠标


## 目标
为手部不便的残障人士提供使用鼠标的遍历，理论上，能控制鼠标即可使用软键盘来控制电脑。

## 原计划
	arduino uno+三轴加速计：实现传感器姿态检测，来移动鼠标
	同时连接使用压电陶瓷震动传感器实现点击等功能

## 原计划废置原因
	比赛方提供套件缺少三轴加速计，无法基于arduino实现

## 实际方案
	改用自带STC-B型开发板基于15F2K61S2型51单片机实现鼠标移动检测

## 开发进度
	由于单片机开发难度较高，目前项目产生了比较麻烦的bug，两个版本中，“重力”文件夹中工程可以顺利和上位机通信并自动矫零，但是传入数据不正确
	“重力2”中工程可以实现角度的显示但是无法进行串口通信
	使用python开发了串口检测模块，能够得到四个元素[x1,x2,y1,y2]的角度信息
	“重力”中get.py是用例模块，try.py是使用样例

我们期望采用三轴加速计模块和开发板模块独立出来，能够采用类似外设的方式来实现绑在腿或头部来控制鼠标，开发板在口袋中（若使用arduino mini会更方便）

这样能检测运动来实现类似体感控制的效果，在手部残障人士，惯用右手骨折等环境中可以作为辅助设备，同时，若能进一步提高精度，则可以用于游戏外设或者代替触摸板来工作
